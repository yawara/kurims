4. $f = x^2+y^2$に対して(16)を確認せよ¶

$\frac{\partial f}{\partial r} = \cos \theta \frac{\partial f}{\partial x}+\sin \theta \frac{\partial f}{\partial y}$について¶

$f = r^2$より、

（左辺）$= 2r$

$x = r \cos \theta, y= r \sin \theta$より、

（右辺）$= \cos \theta \times ( 2x ) + \sin \theta \times ( 2y ) = = \cos \theta \times ( 2r \cos \theta) + \sin \theta \times ( 2r sin \theta ) = 2r \times ( \cos^2 \theta + \sin^2 \theta ) = 2r$

$\frac{\partial f}{\partial \theta} = -r \sin \theta \frac{\partial f}{\partial x} + r \cos \theta \frac{\partial f}{\partial y}$について¶

$f$は$\theta$に依存していないから、

（左辺）$= 0$

$x = r \cos \theta, y= r \sin \theta$より、

（右辺）$= -y \frac{\partial f}{\partial x} + x \frac{\partial f}{\partial y} = -y \times (2x) + x \times ( 2y ) = 0$
